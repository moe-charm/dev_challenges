# NyanZip

> A delightfully useless cat-language compression app for the DEV April Fools Challenge.

## Live Demo

- [Open NyanZip](https://moe-charm.github.io/dev_challenges/20260402april/)

## What It Does

NyanZip takes normal text and turns it into a cat-flavored mess of `meow` and `MEOW!!`.

- Playful compression mode for maximum nonsense
- Ultra mode with real compression under the hood
- Optional Gemini-powered cat commentary
- Japanese and English UI

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages
- Google Gemini API

## Project Structure

- `index.html` - repository landing page and redirect to the app
- `20260402april/index.html` - main app UI
- `20260402april/app.js` - app logic
- `20260402april/js/engine.js` - cat-text encode/decode core
- `20260402april/js/rans.js` - ultra compression pipeline
- `20260402april/js/chat.js` - optional AI review comments

## Notes

If you open the repository on GitHub, this README gives you the fastest path to the actual app. The root `index.html` also redirects to the app for convenience when hosted on GitHub Pages.
